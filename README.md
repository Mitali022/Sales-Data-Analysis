# Sales Data Analysis System

## 📌 Project Overview

The Sales Data Analysis System is a data analytics project developed to analyze sales performance, identify business trends, and generate meaningful insights through interactive dashboards and visualizations. The project combines web technologies, Python-based data cleaning, and Power BI reporting to transform raw sales data into actionable business information.
